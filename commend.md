
## 시스템 관리 명령어

### 시스템 종료
- **`shutdown -P now`**: 즉시 시스템 종료
- **`halt -p`**: 시스템을 종료하고 전원 끔
- **`poweroff`**: 전원 끄기
- **`init 0`**: 시스템 종료 (Runlevel 0)

### 시스템 재부팅
- **`reboot`**: 시스템 즉시 재부팅
- **`init 6`**: 시스템 재부팅 (Runlevel 6)
- **`shutdown -r now`**: 즉시 재부팅
- **`shutdown -r 10`**: 10분 후 재부팅
- **`shutdown -r 20:00`**: 20:00에 재부팅
- **`shutdown -k +15`**: 15분 후 종료된다고 메시지만 출력 (실제 종료 없음)
- **`shutdown -c`**: 예약된 종료 또는 재부팅 취소
- **`shutdown -r +5 "system will be rebooted in 5 minutes"`**: 5분 후 재부팅, 메시지 출력
- **`shutdown -P +10`**: 10분 후 시스템 종료

### 로그아웃
- **`logout`**: 현재 사용자 로그아웃
- **`exit`**: 현재 세션 종료
- **`ctrl + d`**: 현재 터미널 세션 종료

## 파일 및 디렉토리 관리 명령어

### 파일 내용 출력
- **`cat 파일명`**: 파일 내용 전체 출력
- **`more 파일명`**: 파일을 페이지 단위로 출력
- **`less 파일명`**: 파일을 페이지 단위로 출력 (더욱 유연한 탐색 가능)
- **`head 파일명`**: 파일 내용 앞부분 10줄 출력
- **`head -3 파일명`**: 파일 내용 앞부분 3줄 출력
- **`tail 파일명`**: 파일 내용 뒷부분 10줄 출력
- **`tail -f 파일명`**: 파일 내용 실시간 출력 (로그 모니터링)
- **`tail -n 10 파일명`**: 파일 내용 뒷부분 10줄 출력
- **`tail -n +10 파일명`**: 파일 내용 10번째 줄부터 출력

### 파일 검색 (grep)
- **`grep "문자열" 파일명`**: 파일에서 문자열 검색
- **`grep -i "문자열" 파일명`**: 대소문자 구분 없이 문자열 검색
- **`grep -v "문자열" 파일명`**: 문자열이 포함되지 않은 라인 출력
- **`grep -c "문자열" 파일명`**: 문자열이 포함된 라인 수 출력
- **`grep -n "문자열" 파일명`**: 문자열이 포함된 라인 번호 출력
- **`grep -r "문자열" 디렉토리명`**: 디렉토리 내 파일에서 문자열 검색
- **`grep -R "문자열" 디렉토리명`**: 디렉토리 내 파일에서 문자열 검색
- **`grep -w "문자열" 파일명`**: 단어 단위 검색
- **`grep -x "문자열" 파일명`**: 전체 라인 검색

## 파일 권한 및 소유자 변경

### 파일 권한 변경 (chmod)
- **`chmod 권한 파일명`**: 파일 권한 변경
- **`chmod 777 파일명`**: 모든 권한 부여 (읽기, 쓰기, 실행)
- **`chmod 755 파일명`**: 소유자는 모든 권한, 그룹과 기타 사용자는 읽기 및 실행 권한만 부여
- **`chmod 700 파일명`**: 소유자만 모든 권한 부여
- **`chmod 666 파일명`**: 모든 사용자에게 읽기, 쓰기 권한 부여 (실행 권한 제외)
- **`chmod u+x 파일명`**: 소유자에게 실행 권한 추가
- **`chmod g-x 파일명`**: 그룹의 실행 권한 제거
- **`chmod o=r 파일명`**: 기타 사용자에게 읽기 권한만 부여

### 파일 소유자 변경 (chown)
- **`chown 소유자명 파일명`**: 파일 소유자 변경
- **`chown 소유자명:그룹명 파일명`**: 파일의 소유자와 그룹 변경
- **`chown -R 소유자명 디렉토리명`**: 디렉토리와 그 하위 파일들의 소유자 변경
- **`chown -h 소유자명 심볼릭링크`**: 심볼릭 링크의 소유자 변경

### 파일 그룹 변경 (chgrp)
- **`chgrp 그룹명 파일명`**: 파일의 그룹 변경
- **`chgrp -R 그룹명 디렉토리명`**: 디렉토리와 그 하위 파일들의 그룹 변경
- **`chgrp -h 그룹명 심볼릭링크`**: 심볼릭 링크의 그룹 변경

## `ls` 명령어: 파일 목록 출력

### 기본 사용법
- **`ls`**: 현재 디렉토리의 파일 목록 출력
- **`ls [경로]`**: 지정된 경로의 파일 목록 출력

### 자주 사용하는 옵션
- **`ls -l`**: 파일의 상세 정보 출력 (긴 형식, 파일 권한, 소유자, 파일 크기, 수정 날짜 등)
- **`ls -a`**: 숨김 파일을 포함하여 모든 파일 출력 (.으로 시작하는 파일 포함)
- **`ls -al`**: 숨김 파일을 포함한 파일의 상세 정보 출력
- **`ls -h`**: 파일 크기를 사람이 읽기 쉬운 형태로 출력 (예: KB, MB, GB 등). `-l` 옵션과 함께 사용하여 파일 크기 확인에 용이
- **`ls -r`**: 역순으로 파일 목록 출력
- **`ls -t`**: 수정 시간 순으로 파일 출력 (최근에 수정된 파일부터)
- **`ls -S`**: 파일 크기 순으로 출력 (가장 큰 파일부터)
- **`ls -d */`**: 현재 디렉토리 내의 하위 디렉토리 목록만 출력

### 파일 종류 및 추가 정보 표시
- **`ls -F`**: 파일 유형에 따른 표시 추가 (/: 디렉토리, *: 실행 파일 등)
- **`ls -i`**: 파일의 inode 번호 출력
- **`ls -Q`**: 파일명을 쌍따옴표로 감싸서 출력
- **`ls -Z`**: SELinux 보안 컨텍스트 정보 출력 (SELinux가 활성화된 시스템에서 사용)

### 파일 필터링 및 정렬 옵션
- **`ls *.txt`**: 특정 확장자를 가진 파일 목록 출력 (예: .txt 확장자)
- **`ls -R`**: 하위 디렉토리까지 재귀적으로 모든 파일과 디렉토리 출력
- **`ls -X`**: 파일을 확장자 기준으로 정렬하여 출력
- **`ls -1`**: 한 줄에 하나씩 파일 목록 출력 (스크립트에서 유용)

### 파일 정보 요약 및 보기 편하게 출력
- **`ls -m`**: 파일 목록을 쉼표(,)로 구분하여 출력
- **`ls -l --time-style=long-iso`**: 시간을 YYYY-MM-DD HH:MM 형식으로 출력
- **`ls -l --time-style=full-iso`**: 시간을 전체 ISO 형식으로 출력

- **`ps -ef`**: 모든 사용자의 프로세스 상세 정보 출력
- **`ps -aux`**: 모든 사용자의 프로세스 상세 정보 출력 (BSD 스타일)
- **`ps -e --forest`**: 프로세스 트리 구조로 출력
- **`ps -p 프로세스ID`**: 특정 프로세스 ID에 대한 정보 출력

### kill 명령어

- **`kill 프로세스ID`**: 특정 프로세스 ID에 대한 프로세스 종료
- **`kill -9 프로세스ID`**: 프로세스 강제 종료

### pstree 명령어

- **`pstree`**: 프로세스 트리 출력
- **`pstree -p`**: 프로세스 ID와 함께 트리 출력
